# Contributing to the 2027 Internship List

Thanks for helping students find fresh opportunities.

## Suggest a role

Use the missing-listing issue form. A listing is eligible when it is:

- A software engineering or product management internship or co-op
- Located in the United States or explicitly open to remote U.S. candidates
- Currently accepting applications
- Posted within the last 30 days when an authoritative posting date is available
- Linked directly to the employer or its applicant tracking system

Please provide the direct employer listing, not an aggregator or repost.

## Report a problem

Use the incorrect-listing issue form for closed roles, bad links, duplicates, incorrect dates, or categorization mistakes.

## Pull requests

README.md and data/internships.json are generated automatically and should not be edited by hand. Code or documentation improvements are welcome, but listing changes should be submitted through an issue so the source pipeline remains authoritative.

Posting dates must come from an employer career page, its API, or another authoritative source. We do not copy dates from competing job boards.
